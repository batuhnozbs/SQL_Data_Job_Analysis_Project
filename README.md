# SQL Data Analyst Job Market Analysis

## Overview

This project explores the 2023 Data Analyst job market using SQL and PostgreSQL. The analysis focuses on identifying high-paying jobs, the skills employers value most, and the technologies that provide the best balance between salary and market demand.

The dataset includes thousands of job postings with salary information, company details, and required technical skills.

## SQL Queries

### 1. Top Paying Jobs

**File:** `1_top_paying_jobs.sql`

Finds the highest-paying remote Data Analyst jobs with available salary information. The query joins company information to display which organizations offer the best salaries.

---

### 2. Skills Required for Top Paying Jobs

**File:** `2_top_paying_job_skills.sql`

Analyzes the skills requested in the highest-paying Data Analyst positions by combining job postings with skill data. This helps identify which technical skills are common among premium salary roles.

---

### 3. Most Demanded Skills

**File:** `3_top_demanded_skills.sql`

Calculates the most frequently requested skills across remote Data Analyst job postings. The results highlight the technologies employers expect candidates to know.

---

### 4. Highest Paying Skills

**File:** `4_top_paying_skills.sql`

Computes the average salary associated with each skill to determine which technologies are linked to higher-paying Data Analyst positions.

---

### 5. Optimal Skills

**File:** `5_optimal_skills.sql`

Combines salary and demand metrics to identify the most valuable skills to learn. The query ranks skills based on both average salary and market demand.
